# Delegated Coding — Managing your AI collaborators

### ⬇ [Download the slides (PDF)](https://github.com/annawoodard/delegated-coding/raw/main/slides.pdf)

A talk for UChicago DSI Summer Lab / AI+Science REU students on using AI coding
agents in research: how to delegate work you can still vouch for, what to check,
and what stays yours.

By [Anna Woodard](https://github.com/annawoodard) — UChicago Data Science
Institute · UChicago Medicine.

---

The slides are written in [Quarto](https://quarto.org) (reveal.js). To rebuild:

```
micromamba run -n quarto quarto render slides.qmd
```

Companion skills referenced in the talk:
[uchicago-dsi/ai-sci-skills](https://github.com/uchicago-dsi/ai-sci-skills).
